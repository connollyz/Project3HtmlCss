gaia coffee house ( student project )

Gaia was chosen as the name for an organic coffee shop. Gaia meaning life in Greek. Element type face was designed for organic packaging giving it a airy natural feel, pared with Oriya Sangam MN a more simple font to add balance to the design.

Element is a transitional font inspired by Baskerville and Harrington. Element is designed specifically for the purpose of food packaging, an industry with vast growth creating many opportunities for new packaging design. The organic theme of Element was achieved by designing with no harsh edges or corners making it flow nicely throughout the typeface. Friendly swashes were added to create character and help keep that light organic feeling. Recommended point size for Element is 50pt.
